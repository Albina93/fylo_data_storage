# Fylo Data Storage Component

The goal was to recreate the Figma design using HTML and Bootstrap for layout and styling.
The challenges I encountered during the project is matching the exact Figma spacing and sizes while still using Bootstrap's default paddings and margins. Making the background pattern and containers line up correctly on different screen sizes while staying inside the bootstarp layout.
My approach was to first structure the layout with HTML, then applying Bootstrap utility classes, and finally add custome CSS for color, font size etc.
If I would've given more time, I would deepen my knowledge of Bootstrap to write cleaner and more efficent layouts.
